    Enumeration StandardListTokens

    L'énumération StandardListTokens définit un ensemble de jetons standard pour différents types de listes. Chaque jeton est construit en combinant le préfixe « LST » avec un identifiant unique pour le type de liste. Par exemple, le jeton de la liste « Actions » est construit sous la forme ${Prefix.list}0000-000-001.${Prefix.action}. Cette énumération peut être utilisée pour référencer facilement ces jetons de liste standard dans le code.
