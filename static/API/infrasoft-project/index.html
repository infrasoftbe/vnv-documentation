Infrasoft Project / Exports

Ce référentiel contient des classes et des interfaces TypeScript liées à la gestion de nœuds, de relations, de métadonnées, de structures et d'opérations dans un système de gestion de données. Voici une brève description des principaux éléments du code :

Classes Principales

1. OperationStack

La classe OperationStack est une structure de données qui stocke et gère une collection d'opérations à l'aide d'un objet Map. Les opérations peuvent inclure des actions telles que l'ajout, la mise à jour ou la suppression d'éléments.

2. NodeExtended

La classe NodeExtended étend la fonctionnalité de la classe Node en ajoutant des méthodes pour manipuler les nœuds, les relations, les métadonnées, les structures et les opérations. Elle intègre également les fonctionnalités de la classe OperationStack pour gérer les opérations associées.

3. MetaContainer

L'objet MetaContainer représente une métadonnée qui contient des informations supplémentaires liées à un nœud, une relation ou une structure.

4. Node

La classe Node représente un nœud dans la structure de données, avec des propriétés telles que self et data.

5. NodeData

La classe NodeData représente les données associées à un nœud, avec des méthodes pour ajouter, mettre à jour, supprimer et interroger des informations sur le nœud.

6. Operation

La classe Operation représente une opération effectuée sur les données, avec des propriétés telles que data et eventTime.

Interfaces

1. IOperationStack

L'interface IOperationStack décrit la structure d'une pile d'opérations, spécifiant les méthodes telles que add, get, update, delete, etc.

2. INodeExtended

L'interface INodeExtended décrit la structure d'un nœud étendu avec des méthodes supplémentaires pour manipuler les nœuds, les relations, les métadonnées, les structures et les opérations.

3. INodeExtendedInitOptions

L'interface INodeExtendedInitOptions décrit les options d'initialisation d'un nœud étendu, permettant de spécifier des paramètres tels que self, data, et operations.

4. TOperationStackOptionsInit

Le type TOperationStackOptionsInit décrit les options d'initialisation d'une pile d'opérations, notamment les opérations initiales à charger dans la pile.

5. NodeListInitOptions

L'interface NodeListInitOptions décrit les options d'initialisation d'une liste de nœuds, spécifiant les propriétés telles que nodeType et nodes.
