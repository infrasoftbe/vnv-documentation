Enumeration Prefix

Le bloc de code définit une énumération appelée « Préfixe » qui représente les valeurs de préfixe pour différents types de nœuds dans un jeton de nœud. Chaque type de nœud se voit attribuer une valeur de préfixe correspondante. Par exemple, le type de nœud « commande » reçoit la valeur de préfixe « PO », « livrable » reçoit « DEL », et ainsi de suite. Cette énumération peut être utilisée pour générer des jetons de nœud en combinant le préfixe avec un identifiant unique.
